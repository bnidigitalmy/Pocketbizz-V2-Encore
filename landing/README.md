# PocketBizz Landing (Next.js + Tailwind)

Landing page BM untuk PocketBizz. Fokus SME & F&B dengan CTA “Mula Free Trial 7 Hari”.

## Jalankan Lokal
```
cd landing
npm install
npm run dev
```

## Deploy ke Vercel
- Import projek/folder ini ke Vercel.
- Tiada env diperlukan (static).
- Set domain custom: `pocketbizz.my` (Vercel → Settings → Domains).

## Struktur Seksyen
- Hero + CTA
- Pain points
- Features (Inventori, Konsainan, Claims, Reports, Multi-channel, RBAC)
- Flow ringkas
- Pricing (Pro RM39, Early Adopter RM29, 6/12 bulan diskaun)
- Testimoni placeholder
- Screenshot placeholder
- FAQ
- CTA penutup
